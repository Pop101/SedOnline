Has a response ever not been just right?
Run find-and-replace on your requests today!

q

Usage

Get /<pattern>/<replacement>/<input>

pattern is a regular expression.
replacement is a string.
input is a string or URL

But Why?

Syncing calendars into Google Calendar is easy with iCal links, but what if your source calendar is missing critical info? That happened to me, and I wasn't allowed to add a location afterwards. So I made this tool to add it for me.
